# Fanarts-Site
This is a simple site that shows different fanarts. You can publish a fanart, see other fanarts and much more...

**Unaauthorized users**

Users who are unauthorized can access:
 -Catalog page
 -Details pages
 -Login page
 -Register page
 
 
 **Authorized users**
 
 Users who are unauthorized can access::
  -Catalog page
  -Details pages
  -Create fanart
  -Logout functionallity
